# Video Game Sales Analysis and Marketing Strategy (2017)

## English Version

### Project Description

This project analyzes historical video game sales data to identify the key factors that drive commercial success in the gaming industry. The analysis is designed to support strategic marketing decisions for 2017 based on trends observed up to 2016.

The dataset includes information on regional sales, platforms, genres, user and critic reviews, and ESRB ratings.

### Objectives

- Identify platforms with the highest growth potential
- Determine the most profitable game genres
- Analyze regional differences in user preferences
- Evaluate the impact of user and critic reviews on sales
- Test statistical hypotheses related to user ratings

### Methodology

#### 1. Data Preparation

- Data cleaning and standardization
- Handling missing values and special cases, such as `"TBD"`
- Data type conversion
- Calculation of total global sales

#### 2. Exploratory Data Analysis

- Trends in game releases over time
- Platform life cycles and performance
- Genre profitability analysis
- Correlation between reviews and sales

#### 3. Regional Analysis

- Most popular platforms by region: North America, Europe, and Japan
- Genre distribution across regions
- Impact of ESRB ratings on regional sales

#### 4. Hypothesis Testing

- Comparison of user ratings between Xbox One and PC
- Comparison of user ratings between Action and Sports genres

### Key Insights

- Identification of high-performing platforms and genres
- Regional differences in gaming preferences
- Insights into how reviews influence sales performance
- Data-driven recommendations for marketing strategy

### Outcome

The analysis provides actionable insights to guide advertising investment and business strategy, helping maximize returns in the 2017 video game market.

## Versión en Español

### Descripción del Proyecto

Este proyecto analiza datos históricos de ventas de videojuegos con el objetivo de identificar los factores que determinan el éxito comercial en la industria. El análisis está orientado a apoyar la planificación estratégica de campañas de marketing para el año 2017, utilizando información disponible hasta 2016.

El conjunto de datos incluye ventas por región, plataformas, géneros, reseñas de usuarios y críticos, así como clasificaciones ESRB.

### Objetivos

- Identificar plataformas con mayor potencial de crecimiento
- Determinar los géneros más rentables
- Analizar diferencias en preferencias por región
- Evaluar la influencia de reseñas en las ventas
- Probar hipótesis estadísticas sobre calificaciones de usuarios

### Metodología

#### 1. Preparación de Datos

- Limpieza y estandarización de datos
- Tratamiento de valores ausentes y casos especiales, como `"TBD"`
- Conversión de tipos de datos
- Cálculo de ventas globales

#### 2. Análisis Exploratorio de Datos

- Evolución de lanzamientos en el tiempo
- Ciclo de vida de plataformas
- Rentabilidad por género
- Relación entre reseñas y ventas

#### 3. Análisis Regional

- Plataformas más populares por región: Norteamérica, Europa y Japón
- Distribución de géneros por región
- Impacto de clasificaciones ESRB en ventas

#### 4. Pruebas de Hipótesis

- Comparación de calificaciones entre Xbox One y PC
- Comparación entre géneros Acción y Deportes

### Principales Hallazgos

- Identificación de plataformas y géneros más exitosos
- Diferencias claras en preferencias regionales
- Influencia de reseñas en el rendimiento de ventas
- Recomendaciones estratégicas basadas en datos

### Resultado

El proyecto proporciona conclusiones accionables para optimizar la inversión publicitaria y mejorar la toma de decisiones estratégicas en el mercado de videojuegos en 2017.
